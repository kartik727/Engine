Seldonian Engine documentation
=============================================
Welcome to the documentation for the Seldonian Engine, the core library for implementing `Seldonian algorithms <http://aisafety.cs.umass.edu/overview.html>`_. The source code is hosted on `GitHub <https://github.com/seldonian-framework/Engine>`_.

If you are completely new to this library, see the :doc:`overview`.
