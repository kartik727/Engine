Seldonian Engine documentation
=============================================
Welcome to the documentation for the Seldonian Engine, the core library of the `the Seldonian Machine Learning Toolkit <https://seldonian.cs.umass.edu/>`_. The source code is hosted on `GitHub <https://github.com/seldonian-framework/Engine>`_.

If you are completely new to this library, see the :doc:`overview`.
