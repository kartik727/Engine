seldonian.RL.environments.gridworld3x3.Environment¶

class Environment¶

Bases: object

__init__()¶

3x3 gridworld environment (states: 0-8) with a reward function of 0 everywhere except r=-1 in cell 7 (bottom middle) and r=1 in cell 8 (bottom right). Actions are 0,1,2,3 for up,down,left,right respectively. An episode starts in cell 0 (top left) and terminates in cell 8 or if timeout occurs.

Variables:

  • gamma (float) – Discount factor when calculating returns

  • timeout (int) – The number of timesteps after which we terminate the agent regardless of state

  • states (numpy ndarray(int)) – The possible states

  • actions (numpy ndarray(int)) – The possible actions

  • reward_dict (dict) – Maps state, action pair to reward

  • environ_dict (dict) – Maps state, action pair to next state

  • initial_state (int) – Where the agent starts each episode

  • terminal_state (int) – The terminal state

  • current_state (int) – The current state of the agent

  • initial_weights (numpy ndarray) – The initial parameter weights of the agent’s policy model

  • param_weights (numpy ndarray) – The current parameter weights of the agent’s policy model
