interface.cli_supervised.run_interface¶

run_interface(data_pth, metadata_pth, include_sensitive_columns=False, include_intercept_term=False, save_dir='.')¶

Runs the command line interface

Parameters:

  • data_pth – Path to main dataset file

  • metadata_pth – Path to metadata JSON file

  • include_sensitive_columns – Whether to include sensitive columns during training/prediction

  • include_intercept_term – Whether to add a column of ones as the first column in the dataset.
