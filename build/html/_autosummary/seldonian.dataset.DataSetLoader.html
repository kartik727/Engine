class DataSetLoader(regime, column_names, sensitive_column_names=[], include_sensitive_columns=False, include_intercept_term=False, **kwargs)¶

Bases: object

__init__(regime, column_names, sensitive_column_names=[], include_sensitive_columns=False, include_intercept_term=False, **kwargs)¶

Object for loading datasets from disk into DataSet objects

Parameters:

  • regime (str) – The category of the machine learning algorithm, e.g. supervised or RL

  • column_names (List(str)) – list of all column names in the dataset.

  • sensitive_column_names (List(str)) – The names of the columns that contain the sensitive attributes.

  • include_sensitive_columns – Whether to include sensitive columns during training/prediction (supervised learning).

  • include_intercept_term – Whether to add a column of ones as the first column in the dataset (supervised learning).

Variables:

label_column (str) – The column with the target labels (supervised learning)
