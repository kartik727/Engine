seldonian.spec.RLSpec¶

class seldonian.spec.RLSpec(dataset, model_class, frac_data_in_safety, primary_objective, initial_solution_fn, parse_trees, RL_environment_obj, use_builtin_primary_gradient_fn=True, custom_primary_gradient_fn=None, assign_delta_weight_method='equal', bound_method='ttest', optimization_technique='gradient_descent', optimizer='adam', optimization_hyperparams={}, regularization_hyperparams={}, normalize_returns=False)¶

Bases: Spec

Specification object for running RL Seldonian Algorithm

Parameters:

  • model_class – The model class for the Seldonian model, not an instance of the model

  • use_builtin_primary_gradient_fn (bool) – Whether to use the built-in function for the gradient of the primary objective, if one exists. If False, uses autograd

  • custom_primary_gradient_fn (function taking data and model weights and outputting the derivative of the primary objective w.r.t model weights as a scalar) – A function for computing the gradient of the primary objective. If None, falls back on builtin function or autograd
